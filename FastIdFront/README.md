# FastIdFront

Angular SPA для публичного канала FastID (оформление, QR, расчёт премии).

API вызывается через same-origin BFF: `/api/public/...`

## Dev

```bash
npm ci
npm start
```

Приложение: http://localhost:4201

Для работы API нужен запущенный FastIdAPI или docker compose из корня проекта FastID.
